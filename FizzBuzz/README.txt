Usage: .\FizzBuzz.exe Number
If the user does not use this usage, user will be reminded of the proper usage.

Command to compile: gcc main.c -o FizzBuzz

This program takes a range of number from 0 to n, which is supplied by the user.
If the number is divisible by 7, it will print "[*]FIZZ"
If the number is divisible by 3, it will print "[*]BUZZ"
If the number is divisible by 7 and 3, it will print "[*]FIZZBUZZ"
If the number is not divisible by 7 or 3, it will just print that number
