This Code Takes a text file and Copies it onto another file. While copying, the program keeps track of the copying progress in the form of a perentage while also displaying a live representation of the progression in the terminal.

Command Used to compile the code: gcc copyProgression.c -o Copier

Usage dependencies: .\Copier.exe src_file dst_file

If the input command in the terminal does not follow the correct usage, the program will output the exact text that I have given to inform the user on the proper usage command.
